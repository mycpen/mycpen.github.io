# 基于 Hexo 搭建的私人博客
## 一. 技术支持
1. [Hexo](https://hexo.io/zh-cn/)
   - Hexo 是一个快速、简洁且高效的博客框架。
   - Hexo 使用 [Markdown](http://daringfireball.net/projects/markdown/)（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

2. [Pure](https://github.com/cofess/hexo-theme-pure)
   - Hexo-Theme-Pure 是一个 Hexo 的主题框架。

3. [Valine](https://valine.js.org/)
   - 一款快速、简洁且高效的无后端评论系统。
   - 特性：快速；安全；Emoji 😉；无后端实现；MarkDown 全语法支持；轻量易用；文章阅读量统计。
