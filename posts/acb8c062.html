参数解释：

  • cron：定义工作流的时间表（有延迟），为格林尼治时间（UTC），换算为北京时间需在该基础上增加 8 小时，参考：Actions 中的 cron 格式和设置、Actions 时区及缓存问题处理

  • UPYUNUSERNAME, UPYUNPASSWORD, GITHUBUSERNAME, GITHUBEMAIL，为自定义的 Secrets 变量

    变量名释义
    UPYUNUSERNAME又拍云 FTP 用户名，格式「操作员名/服务名」，参考 又拍云视频教程、又拍云文档
    UPYUNPASSWORD又拍云 FTP 密码，参考同上
    GITHUBUSERNAMEGitHub 账号，如 mycpen
    GITHUBEMAILGitHub 账号绑定的主邮箱

  • ftp.py：Python 脚本，使用 FTP 同步云存储与仓库的文件内容

  • git.sh：将本地变更推送至远端的脚本，git add, git commit, git push 这类
